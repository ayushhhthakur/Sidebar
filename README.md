## How to Use

1. Clone the repository:

    ```bash
    git clone https://github.com/ayushhhthakur/Sidebar.git
    ```

2. Open the `index.html` file in your browser.

3. Customize the sidebar by modifying the HTML, CSS, and JS files.

- Responsive design
